Snipt API Contest

Build an app that uses the new Snipt API and be entered to win two conference passes to An Event Apart of your choosing (worth $1,800+). Public voting determines the winner.

Rules

  • App must be complete by Sunday, April 1st, 2012 at 11pm EST.
  • The app must use some part of the public or private Snipt APIs.
  • App can be for desktop, mobile, CLI, etc. Anything goes.

API

The API is finished on our end, but we need to get the site up on Snipt.net first so you can use real endpoint URLs, API keys, etc. The authentication mechanism is API-key (no OAuth). Data formats available are JSON, JSONP, XML, YAML.

We will be launching the new Snipt sometime before March 1st, 2012. This will give you at least one full month to build the app.

The private APIs, once launched, will give you complete read/write access to a user's snipts.

Some sample public endpoints:

  • http://beta.snipt.net/api/public/?format=json
  • http://beta.snipt.net/api/public/snipt/?format=json
  • http://beta.snipt.net/api/public/snipt/30532/?format=json
  • http://beta.snipt.net/api/public/tag/?format=json
  • http://beta.snipt.net/api/public/snipt/?tag=20&format=json
  • http://beta.snipt.net/api/public/user/3/?format=json

Teams

  • Feel free to partner up on this project, if you'd like. There are no restrictions on the number of team members participating, but there are only two tickets for prizes (see below).
  • One entry = one application (regardless of the number of team members)

Prize

  • Once your app is complete and submitted, it will appear in our app showcase. From there, anyone can vote for your app to win after Monday April 2nd at 11am EST, . The app with the most votes by Monday, April 9th at 11am EST will be the winner (one full week of voting).
  • The winner will receive two tickets to An Event Apart of your choosing (event must not be sold-out). Each ticket includes access to the 2-day conference but *not* the workshop.

Questions?

Leave a comment below and we'll respond on this page. Also, follow @Snipt on Twitter.
